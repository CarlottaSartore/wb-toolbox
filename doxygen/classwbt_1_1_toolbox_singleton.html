bool ToolboxSingleton::storeConfiguration ( const std::string &  confKey,
const Configuration &  config 
)

Saves in the singleton a new configuration config.

If the config is valid and hasn't been already stored, it creates a new entry in ToolboxSingleton::m_interfaces. If a configuration with matching confKey is found, if the Configuration object is the same it does nothing, otherwise it overrides it.

Note
Since confKey is the name of the block, the overriding can happen only after subsequent compilation in Simulink, when the block's parameters have been changed.
Parameters
confKeyThe key describing the configuration (name of the Simulink block)
configThe wbt::Configuration object parsed from Simulink's parameters
Returns
Returns true if configure is successful, false otherwise
