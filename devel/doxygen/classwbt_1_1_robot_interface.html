#include <RobotInterface.h>

Detailed Description

Class for handling model and robot resources.

This class is a wrapper of yarp::dev::RemoteControlBoardRemapper and iDynTree::KinDynComputations. By filling information in a wbt::Configuration object, it provides a simple initialization and combined usage of these two resources.

const std::shared_ptr< ControlBoardIndexLimit > RobotInterface::getControlBoardIdxLimit ( )

Get the map between the ControlBoard index inside the RemoteControlBoardRemapper and the number of the controlled joints belonging to it.

Remarks
For example, if the joints are ``` j1_cb1 j2_cb1 j3_cb1 j1_cb2 j2_cb2 j1_cb3 ``` the generated map is ``` {{0,3}{1,2}{2,1}} ``` Note that the map key is 0-indexed.
Returns
The control board limit map.
const std::shared_ptr< JointNameToIndexInControlBoardMap > RobotInterface::getControlledJointsMapCB ( )

Get the map between model joint names and the index representing their relative ordering inside the controlledJoints vector relative to their ControlBoard.

Remarks
For example, if the joints are j1_cb1 j2_cb1 j3_cb1 j1_cb2 j2_cb2 j1_cb3, the map links them to 0 1 2 0 1 0. Note that joints are 0-indexed.
Returns
The joint map.
