Detailed Description

template<typename T>
class wbt::Parameter< T >

Class for storing a generic parameter.

A generic parameters can be either a scalar or a vector. Supported types are defined by the wbt::ParameterType enum.Use wbt::ParameterMetadata to set these information.

Template Parameters
Thetype of the container type. For vector parameters, T is the type of an element of the container.
